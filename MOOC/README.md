# 中国大学慕课AI答题助手

一个基于Tampermonkey的油猴脚本，用于自动提取中国大学慕课(icourse163.org)页面的题目信息并调用AI接口辅助作答。

## 功能特性

- 自动识别题目类型（单选、多选、判断、填空、问答）
- 自动提取题干与选项内容
- 调用OpenAI兼容API获取答案
- 答案展示在题目旁边
- 支持自动点击选项
- 支持多种页面格式
- 可配置调试模式查看请求/响应

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 创建新脚本，将 `mooc-ai-helper.user.js` 内容粘贴进去
3. 修改脚本中的 `API_CONFIG` 配置项
4. 保存脚本并启用

## 配置说明

```javascript
// ========== 配置项 ==========
const CONFIG = {
  showDebug: true,    // 是否显示调试信息（请求/响应内容）
  autoClick: true,    // 是否自动点击选项
  delay: 100,         // 每题之间的延迟(ms)
};

// API配置
const API_CONFIG = {
  baseUrl: "https://api.example.com/v1/chat/completions",  // API地址
  apiKey: "sk-your-api-key",                                // API密钥
  model: "gpt-3.5-turbo",                                   // 模型名称
};
```

## 使用方法

1. 打开中国大学慕课的测验/考试页面
2. 等待页面完全加载
3. 点击右下角的「AI答题」按钮
4. 等待AI处理完成

## 支持的页面格式

- **格式1**: 表单式题目页面（ant-design样式）
- **格式2**: 列表式题目页面（u-questionItem样式）
